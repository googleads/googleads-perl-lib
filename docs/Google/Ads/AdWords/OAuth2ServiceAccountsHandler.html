NAME

Google::Ads::AdWords::OAuth2ServiceAccountsHandler

Back to Top

DESCRIPTION

A concrete implementation of the Google::Ads::Common::OAuth2ServiceAccountsHandler manpage that defines the scope required to access AdWords API servers using OAuth2 for Service Accounts, see <https://developers.google.com/accounts/docs/OAuth2> for details of the protocol.

Refer to the base object the Google::Ads::Common::OAuth2ServiceAccountsHandler manpage for a complete documentation of all the methods supported by this handler class.

Back to Top

ATTRIBUTES

Each of these attributes can be set via the constructor as a hash. Alternatively, there is a get_ and set_ method associated with each attribute for retrieving or setting them dynamically.

server

The API server path to use to construct the scope, defaults to https://adwords.google.com and be changed to https://adwords-sandbox.google.com for use with the Sanbox environment.

Back to Top

METHODS

_scope

Method defined by the Google::Ads::AdWords::AuthTokenHandler manpage and implemented in this class as a requirement for the OAuth2 protocol.

Returns

Returns the scope used to generate access tokens.
