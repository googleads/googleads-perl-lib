NAME

Google::Ads::Common::MapUtils

Back to Top

SYNOPSIS

 use Google::Ads::Common::MapUtils;
 my $map_entries = Google::Ads::Common::MapUtils::get_map($map_xml);
 # Make use of $map_entries.

Back to Top

DESCRIPTION

A collection of utility methods for working with maps (associative arrays).

Back to Top

SUBROUTINES

get_map

Gets a map (associative array) from an array of map entries. A map entry is any object that has a key and value field.

Parameters

An array of map entries.

Returns

A map built from the keys and values of the map entries.

build_api_map

Builds an API map as defined in the WSDLs, which is really a list of key-value maps from a given native Perl map.

Parameters

A native Perl map to be translated to "map" as expected by the API.

Returns

A list of maps in which each map represents an entry of a key and value from the given map.
