NAME

Google::Ads::Common::HTTPTransport - Specialization of the SOAP::WSDL::Transport::HTTP manpage transport class with added logic to handle OAuth.

Back to Top

DESCRIPTION

Provides a thin transport class on top of the SOAP::WSDL::Transport::HTTP manpage to add support to pluggable authorization methods.

Back to Top

ATTRIBUTES

client

Holds an instance of the API client, which will use to retrieve the current the Google::Ads::Common::AuthHandlerInterface manpage.

Back to Top

METHODS

send_receive

Overrides the SOAP::WSDL::Transport::HTTP manpage send_receive method to change the endpoint in case OAuth is enabled, uses the the Google::Ads::AdWords::Client manpage configured OAuth handler to leverage all the OAuth signing logic.

Parameters

The same as the the SOAP::WSDL::Transport::HTTP manpage send_receive method.
